[IES008] Estruturas de Dados

Professor: Willian Guimarães Borges

Sobre a Matéria

Estruturas de Dados é uma disciplina fundamental na ciência da computação, focada em como os dados são organizados, armazenados e manipulados de forma eficiente para otimizar o desempenho de algoritmos. Nesta matéria, exploramos desde os tipos de dados básicos até estruturas complexas como listas, pilhas, filas e árvores, aprendendo a escolher a estrutura correta para cada problema, visando a eficiência em tempo e espaço.

Tópicos Abordados

Revisão Tipo de Dados

Revisão dos tipos de dados primitivos (inteiro, float, char, boolean) e sua importância na alocação de memória e na definição do comportamento das variáveis dentro de um programa.

Estrutura Condicional

Estudo das estruturas de controle de fluxo (if, else if, else) que permitem ao programa tomar decisões e executar diferentes blocos de código com base em condições lógicas.

Laço de Repetição

Abordagem de estruturas de repetição (for, while, do-while) que permitem a execução de um bloco de código múltiplas vezes, essencial para processar coleções de dados e realizar tarefas repetitivas.

Funções

Criação e utilização de funções para modularizar o código, promover a reutilização, organizar programas complexos e melhorar a legibilidade e manutenção.

Ponteiros

Estudo de ponteiros, variáveis que armazenam endereços de memória de outras variáveis, permitindo manipulação direta da memória e implementação de estruturas de dados dinâmicas.

Vetor - Matriz 1D

Implementação e manipulação de vetores (arrays unidimensionais) para armazenar coleções de elementos do mesmo tipo de forma contígua na memória, acessados por índices.

Matriz 2D

Trabalho com matrizes bidimensionais, compreendendo sua representação na memória e como utilizá-las para armazenar dados em formato de tabela, como grades ou imagens.

Struct

Definição e uso de structs (estruturas) para agrupar diferentes tipos de dados em uma única unidade lógica, facilitando a representação de entidades complexas.

Lista Encadeada

Compreensão e implementação de listas encadeadas, uma estrutura de dados dinâmica onde os elementos são nós que contêm dados e um ponteiro para o próximo nó, permitindo inserções e exclusões eficientes.

Pilhas e Filas

Estudo de Pilhas (LIFO - Last In, First Out) e Filas (FIFO - First In, First Out), suas operações básicas (push/pop para pilhas, enqueue/dequeue para filas) e suas aplicações em algoritmos e sistemas.

Fila com Prioridade

Implementação e uso de filas com prioridade, onde os elementos são removidos com base em sua prioridade, em vez da ordem de chegada, comum em sistemas operacionais e simuladores.

Fila com Vetor

Implementação de filas utilizando vetores, abordando os desafios de redimensionamento e o uso de "fila circular" para otimizar o espaço.

Listas Duplamente Encadeadas

Análise de listas duplamente encadeadas, onde cada nó possui ponteiros para o próximo e para o anterior, permitindo travessia em ambas as direções e remoções mais eficientes.

Fila Circular

Exploração da fila circular, uma variação da fila que utiliza um arranjo circular para otimizar o uso do espaço, eliminando a necessidade de movimentar elementos após remoções.

Lista Duplamente Encadeada Circular

Estudo da lista duplamente encadeada circular, combinando as características de ambas: elementos com ponteiros para frente e para trás, e o último elemento apontando para o primeiro, formando um ciclo.

Curiosidades

  • A escolha da estrutura de dados correta pode ser mais importante do que a velocidade do processador para a performance de um programa.
  • Pilhas são usadas para gerenciar chamadas de funções em linguagens de programação (call stack) e para desfazer operações (undo).
  • Filas são essenciais em sistemas de impressão, processamento de tarefas em background e simulações de eventos.

[IND053] Técnicas Avançadas de Programação Web e Mobile

Professor: Marcelo Tadeu Boer

Projeto desenvolvido: Repositório do site desenvolvido

Sobre a Matéria

Esta disciplina explora o desenvolvimento de aplicações web e mobile modernas utilizando frameworks e bibliotecas de ponta. Focamos no React.js para a construção de interfaces de usuário dinâmicas e reativas, e na integração com serviços de backend como o Firebase do Google, capacitando a criação de aplicações completas e com persistência de dados tanto para navegadores quanto para dispositivos móveis.

Tópicos Abordados

Biblioteca React.js

Introdução à biblioteca React.js para construção de interfaces de usuário com base em componentes. Compreensão do Virtual DOM, JSX e o ciclo de vida dos componentes para criar aplicações web eficientes e reativas.

Criação Projeto - Estilização

Passos iniciais para configurar um projeto React, incluindo a estrutura de pastas e a estilização de componentes utilizando CSS, Styled Components ou outros métodos.

Criando Aplicação com React JS e Firebase Google

Desenvolvimento de uma aplicação completa integrando o front-end em React.js com o backend do Firebase do Google, utilizando serviços como Firestore (banco de dados NoSQL), Authentication e Storage.

Atividade Tela Loja

Implementação prática de uma tela de loja/e-commerce, aplicando os conceitos de componentes, gerenciamento de estado e interação com dados externos (como produtos de um banco de dados).

Curiosidades

  • React.js foi criado pelo Facebook e é usado por empresas como Netflix, Airbnb e Instagram, demonstrando sua escalabilidade e poder.
  • Firebase é uma plataforma serverless do Google que oferece diversos serviços de backend (autenticação, banco de dados, hospedagem) que facilitam o desenvolvimento full-stack.
  • O conceito de "componentização" em React permite a reutilização de partes da interface, agilizando o desenvolvimento e tornando o código mais organizado.

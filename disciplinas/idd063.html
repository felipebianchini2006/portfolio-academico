[IDD063] Técnicas Avançadas de Programação

Professor: Wellington Luis Codinhoto Garcia

Sobre a Matéria

Nesta disciplina, mergulhamos em conceitos mais complexos de programação, focando no ecossistema .NET e na linguagem C#. Aprendemos a construir aplicações robustas e escaláveis, explorando a criação de interfaces de usuário com noções de Bootstrap e a manipulação de dados através de operações CRUD (Create, Read, Update, Delete) em um ambiente .NET. É uma base sólida para o desenvolvimento de sistemas mais elaborados.

Tópicos Abordados

Introdução .NET

Visão geral da plataforma .NET, compreendendo sua arquitetura, o Common Language Runtime (CLR), a .NET Framework e .NET Core/5+, e seus benefícios para o desenvolvimento de software.

Conceitos Básicos C#

Revisão e aprofundamento dos fundamentos da linguagem C#, incluindo variáveis, tipos de dados, operadores e a sintaxe básica.

Laços Repetições C#

Estudo de laços de repetição (for, while, do-while, foreach) em C#, essenciais para iterar sobre coleções e executar blocos de código repetidamente.

Conceito Vetores C#

Implementação e manipulação de vetores (arrays) em C# para armazenar coleções de dados do mesmo tipo, compreendendo sua declaração e acesso.

Matrizes C#

Trabalho com matrizes multidimensionais em C#, como matrizes 2D, para representar dados em formato tabular ou grids.

CRUD .NET

Desenvolvimento de operações CRUD (Create, Read, Update, Delete) em aplicações .NET, demonstrando como interagir com bancos de dados para persistir e recuperar informações.

Noção Bootstrap

Introdução ao Bootstrap, um framework de front-end, para estilizar e criar layouts responsivos em aplicações web, proporcionando uma interface de usuário visualmente agradável e adaptável.

Curiosidades

  • O .NET é uma plataforma desenvolvida pela Microsoft que suporta diversas linguagens, sendo C# a mais popular.
  • CRUD é a base de quase todos os sistemas de gerenciamento de dados, desde aplicativos de lista de tarefas até grandes sistemas empresariais.
  • Bootstrap popularizou o design responsivo, tornando mais fácil criar sites que funcionam bem em qualquer dispositivo (celular, tablet, desktop).
